# WTWR

WTWR is a full-stack weather-based clothing app that helps users choose outfits based on the current weather.

## Project Repositories

- Frontend repository: https://github.com/Reggiero31/se_project_react
- Backend repository (Project 13 / se_project_express): https://github.com/Reggiero31/se_project_express

## Features

- View the current weather and suggested outfit conditions
- Sign up, log in, and update your profile
- Add, view, and manage clothing items
- Like items and see them on your profile

## Running the Full-Stack App

1. Clone the frontend repository and the backend repository.
2. Install dependencies in both projects.
3. Start the backend server, then start the frontend app.
4. Open the frontend in your browser to use the app.
